## 📞 Support & Troubleshooting

### For Compilation Issues
1. Run: `mvn clean install`
2. Rebuild project in IDE
3. Check Java version (20+)

### For Visual Issues
1. Clear IDE cache
2. Restart application
3. Check CSS isn't overriding
4. Verify FXML syntax

### For Statistics Not Showing
1. Verify database has books
2. Check statisticsContainer in FXML
3. Ensure refresh() is called
4. Check LivresController imports

### For Performance Issues
1. Reduce animation to 150ms
2. Optimize pie chart size
3. Check system resources
4. Profile application
